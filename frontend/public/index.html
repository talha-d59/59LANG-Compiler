59LANG Quick Reference

Version: 1.0

Syntax:

  • nexus { } - Program entry point
  • shard core x; - Variable declaration (int)
  • shard flux x; - Variable declaration (float)
  • shard sig x; - Variable declaration (bool)
  • shard glyph x; - Variable declaration (string)
  • listen x; - Read input
  • broadcast x; - Print output
  • probe (condition) { } - Conditional (if)
  • fallback { } - Else clause
  • pulse (condition) { } - Loop (while)
  • cycle (...) { } - Loop (for)

Alternate Syntax:

  • main { } - Program entry point
  • var int x; - Variable declaration
  • input x; - Read input
  • output x; - Print output
  • if (condition) { } - Conditional
  • while (condition) { } - Loop

Example:

nexus {
  shard core x, y;
  listen x;
  listen y;
  shard core sum = x + y;
  broadcast sum;
}
